## Cloud Engineer Certification Path

### Primary Certification: AZ-204 — Developing Solutions for Microsoft Azure

**Overview**: This certification validates the ability to design, build, test, and maintain cloud applications and services on Microsoft Azure.

**Recommended Study Pattern**:

- 1–2 hours of focused study daily
- Weekly assessment checkpoints using practice tests
- Target a practice score of 75% or above before booking the real exam
- Total recommended study time: 20 hours over 3–4 weeks

**Core Study Areas**:

1. **Azure Functions** (4 hours)
   - Triggers and bindings
   - Durable functions
   - Deployment slots

2. **Azure Blob Storage** (3 hours)
   - Storage tiers and lifecycle management
   - Shared access signatures
   - Azure Data Lake integration

3. **Azure Cosmos DB** (3 hours)
   - Partitioning strategy
   - Consistency levels
   - SDK usage (Python, .NET)

4. **API Management** (3 hours)
   - Policies and transformations
   - Rate limiting and quotas
   - OpenAPI integration

5. **Azure App Service** (4 hours)
   - Deployment strategies
   - Autoscaling
   - Custom domains and SSL

6. **Azure Cache for Redis** (3 hours)
   - Caching patterns
   - Session state management
   - Cache eviction policies

**Recommended Resources**:

- Microsoft Learn: AZ-204 Learning Path (free, ~20 hours)
- Practice assessments on Microsoft Learn
- John Savill's AZ-204 Study Cram (YouTube)

### Secondary Certification: AZ-305 — Designing Azure Infrastructure Solutions

**Prerequisites**: AZ-204 or equivalent experience

**Overview**: Expert-level certification covering the design of identity, governance, monitoring, data storage, business continuity, and infrastructure solutions.

**Recommended Study Time**: 30 hours over 5–6 weeks

**Core Study Areas**:

- Identity and governance (Azure AD, RBAC, Policy)
- Data storage design (relational, NoSQL, data lake)
- Business continuity (backup, disaster recovery, SLAs)
- Infrastructure (VNets, load balancers, CDN, DNS)
- Migration strategies

## DevOps Engineer Certification Path

### Primary Certification: AZ-400 — Designing and Implementing DevOps Solutions

**Overview**: Validates expertise in working with people, processes, and technologies to continuously deliver business value.

**Recommended Study Time**: 25 hours over 4–5 weeks
**Target Practice Score**: 75% before exam

**Core Study Areas**:

1. **CI/CD Pipelines** (6 hours) — Azure Pipelines, GitHub Actions
2. **Infrastructure as Code** (5 hours) — Bicep, Terraform, ARM templates
3. **Monitoring and Observability** (4 hours) — Azure Monitor, Application Insights
4. **Security in DevOps** (5 hours) — Key Vault, dependency scanning, SAST
5. **Testing Strategies** (5 hours) — Unit, integration, performance testing

## General Study Guidance (All Roles)

**Optimal Study Conditions**:

- Learners with 12–18 meeting hours per week show the best completion rates
- Morning study sessions (pre-9 AM) correlate with higher retention in synthetic outcomes
- Spaced repetition beats marathon sessions: 90 minutes daily > 8 hours on weekends
- Pair each topic with a hands-on Azure lab for 40% better retention

**Assessment Readiness Signal**:

- If practice score is consistently above 75%: ready to book
- If practice score is 65–74%: focus on weak areas for 1 more week
- If practice score is below 65%: return to fundamentals before re-assessing

**Certification Renewal**:

- Most Azure certifications expire after 1 year
- Free renewal assessments available on Microsoft Learn
- Notify manager 60 days before expiry for scheduling support
